🚀 CSE Hub - All-in-One Student Portal
CSE Hub ek full-stack web application hai jo Computer Science ke students ki madad karne ke liye banayi gayi hai. Isme internships find karne se lekar AI-based career guidance tak sab kuch ek hi jagah milega.

✨ Features
💼 Job & Internship Portal: Latest openings browse karein aur apply karein.

🤖 AI Mentor: Ek smart chatbot jo aapke coding aur career se jude sawalon ka jawab deta hai.

📊 Student Dashboard: Apni applications, certificates, aur coding progress ko track karein.

💻 Coding Practice: Platform par hi coding skills ko behtar banayein.

📜 Resume Builder & Hackathons: Career boost karne ke liye tools aur events ki jankari.

🛠️ Tech Stack
Frontend:

Next.js (App Router)

Tailwind CSS (Styling)

Backend:

Node.js & Express.js

MongoDB (Database)

JWT (Authentication)

🚀 Getting Started
Prerequisites
Node.js installed

MongoDB account (Atlas)

Installation
Repository Clone karein:

Bash
git clone https://github.com/pchitranshrao-ui/cse-hub-fullstack.git
cd cse-hub-fullstack
Backend Setup:

Bash
cd backend
npm install
# .env file banayein aur MONGODB_URI, JWT_SECRET dalein
node src/server.js
Frontend Setup:

Bash
cd frontend
npm install
npm run dev
📂 Project Structure
/frontend: Next.js application code.

/backend: Express API aur Database models.

/static-version: Single page HTML/Tailwind demo version.
